剑指 Offer 68 - I. 二叉搜索树的最近公共祖先

给定一个二叉搜索树, 找到该树中两个指定节点的最近公共祖先。

最近公共祖先的定义为：“对于有根树 T 的两个结点 p、q，最近公共祖先表示为一个结点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。”

例如，给定如下二叉搜索树: root = [6,2,8,0,4,7,9,null,null,3,5]

说明:

  • 所有节点的值都是唯一的。
  • p、q 为不同节点且均存在于给定的二叉搜索树中。

个人思路：

因为二叉搜索树一定存在p、q节点，所以并没有做非空校验。

二叉搜索树的是有顺序的，左子树<根节点<右子树

所以这这几种情况：

  • p<=root<=q，return root
  • p<q<root，return method(root.left,p,q)
  • root<p<q，return method(root.right,p,q)

剑指 Offer 68 - II. 二叉树的最近公共祖先

给定一个二叉树, 找到该树中两个指定节点的最近公共祖先。

中最近公共祖先的定义为：“对于有根树 T 的两个结点 p、q，最近公共祖先表示为一个结点 x，满足 x 是 p、q 的祖先且 x 的深度尽可能大（一个节点也可以是它自己的祖先）。”

例如，给定如下二叉树: root = [3,5,1,6,2,0,8,null,null,7,4]

示例 1:

示例 2:

说明:

  • 所有节点的值都是唯一的。
  • p、q 为不同节点且均存在于给定的二叉树中。

个人思路：

个人认为这道题比搜索树还要难一点，关键是返回值的确定，因为搜索树是有序的，而二叉树是无序的，用不到上面的思路。

这道题的思路：p和q的只有两种关系，根节点和子节点、或者是左子树和右子树

对于二叉树中任意两个节点都是这两种关系，即

  • 要么都分别属于不同的子树中
  • 要么其中一个是另一个的祖宗节点。

所以当找到root == p || root == q时，直接就返回，假如在另一个子树没找到另一个值，那么p和q就一定是根节点和子节点关系，所以当前root就是最近公共祖先。

而对于左子树和右子树情况，就判断对于当前root来说，在它的左右子树中找到了p或q没有，如果都没有就返回null，如果都找到就说明root是最近公共祖先，否则是找到一个，就返回找到的值。
