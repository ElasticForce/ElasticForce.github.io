剑指 Offer 45. 把数组排成最小的数

提示:

  • 0 < nums.length <= 100

说明:

  • 输出结果可能非常大，所以你需要返回一个字符串而不是整数
  • 拼接起来的数字可能会有前导 0，最后结果不需要去掉前导 0

思路：

一开始暴力回溯，很明显超时了，需要O(n!)，而这道题n<=100

然后想着分组回溯，按照最高位划分，存在列表里，然后通过map取出对应列表，每个列表取出最小拼接数，最后总拼接，但是想了下好像还是不行，极端条件，比如最高位都为1，最后还是都在一个列表里。
