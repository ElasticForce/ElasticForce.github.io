    494. 目标和

    给定一个非负整数数组，a1, a2, …, an, 和一个目标数，S。现在你有两个符号 + 和 -。对于数组中的任意一个整数，你都可以从 + 或 -中选择一个符号添加在前面。

    返回可以使最终数组和为目标数 S 的所有添加符号的方法数。

    示例：

    提示：

    • 数组非空，且长度不会超过 20 。
    • 初始的数组的和不会超过 1000 。
    • 保证返回的最终结果能被 32 位整数存下。

    思路:

    已知所有数带上符号后，最终结果为S
    将所有带上符号的数分为两部分：正子集x和负子集y，有x+y = S
    而对于数组总和Sum = x-y，所以对于正子集 x = (Sum+S)/2
    这道题就演变为在数组中找到正子集x，满足结果为 (Sum+S)/2 的个数

    完全背包问题

    之前的是01背包，也就是对于一个数，要么装进背包，要么不装进背包，即0或1

    而完全背包，则是每个物品是可以无限的拿取的。

    0/1背包
    定义：物品只能用一次

    dp[i] [j]=max(dp[i-1] [j], dp[i-1] [j-weight[i]] + value[i])

    完全背包
    定义：物品可以用无限次

    dp[i] [j] = max(dp[i-1] [j], dp[i] [j-weight[i]] + value[i])

    你可以认为每种硬币的数量是无限的。

    思路

    这道题可以类比于青蛙跳台阶的那道题，一次跳1层或2层，跳到第n层有多少种跳法，f(n)=f(n-1)+f(n-2)，和这道题很类似。
    比如有1，2，5的面额，需要知道凑出n的最小硬币数，f(n)=min(f(n-1)+1,f(n-2)+1,f(n-5)+1)

    所以对于dp[amount]来说，凑齐amount则需要先凑齐dp[amount-1],dp[amount-2],dp[amount-5]
    然后dp[amount]=min(dp[amount-1],dp[amount-2],dp[amount-5])，以此类推
